# ACML 2026 Overleaf Source Bundle

Upload this folder, or `acml2026_aus_early_warning_overleaf.zip`, to Overleaf.

- Compiler: pdfLaTeX.
- Main paper: set the main document to `main.tex`.
- Supplementary: switch the main document to `supplementary.tex`.

Keep the folder structure unchanged:

- `figures/`
- `tables/`
- `main.tex`
- `supplementary.tex`
- `acml26.bib`
- `jmlr.cls`
